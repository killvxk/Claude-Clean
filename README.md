# claude-clean

Clean up Claude Code system folders.

## Installation

```bash
pip install .
```

Or install in development mode:

```bash
pip install -e .
```

## Usage

```bash
claude-clean
```

## What it cleans

- `~/.claude/projects` - Session files
- `~/.claude/telemetry` - Telemetry data
- `~/.claude/todos` - Todo items
- `~/.claude/tasks` - Task data
- `~/.claude/shell-snapshots` - Shell snapshots
- `~/.claude/session-env` - Session environment
